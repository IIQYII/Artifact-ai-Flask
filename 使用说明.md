a# 文物讲解API测试程序使用说明

## 程序文件
- `simple_test.py` - 简单的交互式测试程序

## 使用方法

### 1. 启动API服务器
```bash
python app.py
```

### 2. 运行测试程序
```bash
python simple_test.py
```

### 3. 按照提示输入信息
```
=== 文物讲解API测试 ===
请输入文物信息进行测试
------------------------------
请输入文物名称（例如：兵马俑）: 青铜鼎
请输入文物朝代（例如：秦朝）: 商朝
```

### 4. 查看结果
程序会显示生成的讲解文案：
```
正在生成 商朝 的 青铜鼎 讲解文案...
✅ 生成成功！
----------------------------------------
文物: 青铜鼎
朝代: 商朝

讲解文案:
----------------------------------------
我是青铜鼎，来自商朝的青铜器...
----------------------------------------
```

## 支持的输入示例
- 兵马俑（秦朝）
- 青铜鼎（商朝）
- 唐三彩（唐朝）
- 青花瓷（明朝）
- 玉璧（周朝）

## 注意事项
- 确保API服务器正在运行
- 确保config.py中的API密钥已正确配置
- 程序支持连续测试多个文物
- 按Ctrl+C可随时退出程序
